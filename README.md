# King Arthur -  Front-End

##  How to Run the Website Locally

You must have **Node.js** and **NPM** installed on your computer.

1.  **Get the code:**
    ```bash
    git clone https://github.com/CarlosDanielSucre/king-arthur-frontend.git
    cd king-arthur-frontend
    ```

2.  **Install the tools:**
    ```bash
    npm install
    ```

3.  **Start the local server:**
    ```bash
    npm run dev
    ```
    The website will open in your browser at a link like `http://127.0.0.1:5173/`.
